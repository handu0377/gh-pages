Toptip

显示在页面顶部的轻量级提示，一般用来反馈用户的操作结果，比如表单校验失败等。

JS方法定义如下：

{% highlight js %} $.toptip(text, [duration, type]); {% endhighlight %}

示例：

{% highlight js %} $.toptip('操作成功', 'success'); $.toptip('操作失败', 'error'); $.toptip('警告', 'warning'); $.toptip('操作成功', 2000, 'success'); //设置显示时间 {% endhighlight %}

注意，此组件从 V0.7.2 版本开始才可以使用。
