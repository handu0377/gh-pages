Count 计数器

从 V1.2.0 开始加入了计数器组件。可以用来显示和修改数值，比如可以用在购物车显示商品数目。

因为计数器的JS逻辑比较简单，所以这里只提供了一个CSS组件，如果你需要写对应的JS代码可以参考如下代码：

{% highlight js %} var MAX = 99, MIN = 1; $('.weui-count__decrease').click(function (e) { var $input = $(e.currentTarget).parent().find('.weui-count__number'); var number = parseInt($input.val() || "0") - 1 if (number < MIN) number = MIN; $input.val(number) }) $('.weui-count__increase').click(function (e) { var $input = $(e.currentTarget).parent().find('.weui-count__number'); var number = parseInt($input.val() || "0") + 1 if (number > MAX) number = MAX; $input.val(number) }) {% endhighlight %}
