帮助 jQuery WeUI

jQuery WeUI 是 MIT 协议的免费开源项目，不会收取任何费用。

作者和大家一样也是一枚程序员，白天上班，业余时间维护这个项目需要耗费不少精力，如果jQuery WeUI 帮到了你，你可以通过以下任意方式来帮她变得更好：

  • 在 github 上给点击 Star 给我加星，更多的星星是我前进的动力。
