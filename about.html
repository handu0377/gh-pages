为什么做 jQuery WeUI

WeUI 目前只包含 CSS 代码，官方短期内不会出JS版本，并且官方组件太少根本不够用，所以我做了一个jQuery版本的方便使用。

jQuery WeUI = WeUI + jQuery + 大量拓展组件

jQuery WeUI 的组件分两部分：

  • 基础组件：weui官方的组件，使用了weui官方的css
  • 拓展组件：jquery weui 专属的组件

有任何Bug或者建议，请在 Github 上提 Issue 给我。或者加入 jQuery WeUI 专属QQ群 546452237.

目前 weui 生态圈已经逐步成型，各种第三方实现的版本都出来了，现在已经有不少公众号在用 weui 进行开发了。如果有正在进行微信公众号开发的童鞋，尽量切换到 weui 吧。

jQuery WeUI 的定位和优势

jQuery WeUI 的定位正如 jQuery 的定位一样：做一把锋利的小刀，而不是做一个笨重的大炮。

所以做一个重量级的框架来解决所有开发问题从来都不是jQuery WeUI 的目的，她的目的很简单：就是想做一个包含一系列常用UI控件的库。

至于模块化，单页应用，项目构建，这些交给更专业的 Vue, Angular, Webpack 来做。

另外我有计划做一个基于 Vue 或者 Angular2 的版本，预计会在下半年，有进展的话会随时通知大家。

jQuery WeUI 会收费吗

jQuery WeUI 是遵循 MIT 协议的免费开源项目，不会收费，我也从不打算靠这个赚钱，纯粹是出于个人兴趣做的。

jQuery WeUI 相关的所有代码都是免费开源的，所以如果在官网上看到的任何内容都可以在我的git项目中找到源码，不要发邮件向我要代码，我没有私自保留任何代码。
